# Simple Note App

Aplikasi catatan sederhana yang dibangun dengan PHP Native untuk tujuan latihan.

## Fitur
- ✅ Buat catatan baru
- ✅ Lihat daftar catatan
- ✅ Edit catatan yang ada
- ✅ Hapus catatan
- ✅ Responsive design
- ✅ Validasi form
- ✅ Pesan feedback

## Teknologi yang Digunakan
- PHP Native
- MySQL
- Bootstrap 5
- Font Awesome
- Vanilla JavaScript
